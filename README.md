# Habitual

**Habitual** is a Gamified Productivity & Wellness Hub designed to make daily tasks and habit building engaging and fun. Built with a modern full-stack architecture, it combines the appeal of gamification with robust task management.

## User Features
- **User Authentication & Profiles:** Secure signup, login, and personalized user profiles.
- **Dashboard Overview:** A centralized hub to track your daily progress, stats, and gamification rewards (XP/Levels).
- **Daily Tasks:** Manage and check off your one-off daily responsibilities.
- **Habits Tracking:** Build routines by tracking recurring habits and monitoring your consistency.
- **Phased Todo Lists:** Break down large tasks into manageable phases.
- **Streaks & Heatmap:** Keep a record of your consecutive days of productivity using visual streak counters and calendar heatmaps.
- **Friends & Social:** Add friends, track each other's progress, and stay motivated together.
- **Gamification:** Earn rewards for completing tasks and habits to level up your profile!

## Tech Stack
- **Frontend:** React (Vite), CSS
- **Backend:** Node.js, Express
- **Database:** MongoDB
