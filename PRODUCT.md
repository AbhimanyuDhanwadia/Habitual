# Habitual — Product Register

## Product Type
Product surface (consumer-facing web application)

## Product Summary
**Habitual** is a Productivity & Wellness Platform that helps users build better habits, manage daily tasks, and track long-term projects. Users maintain consistency streaks, visualize progress through heatmaps and analytics, and discover science-backed habits to integrate into their routines.

## Core Surfaces
- **Landing Page** — Public marketing/conversion page
- **Auth** — Login & signup flows
- **Dashboard** — Authenticated home: stats, daily progress, upcoming todos
- **Daily Tasks** — Checklist-style daily task tracker
- **Habits** — Science-backed habit discovery & tracking
- **To-Do List** — Phased project management with deadlines
- **Account** — Profile, settings, progress stats

## Tech Stack
- Frontend: React (Vite), Vanilla CSS
- Backend: Node.js, Express
- Database: MongoDB
